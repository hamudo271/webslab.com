# WEBSLAB

Marketing site for **WEBSLAB** — a Korean web agency specializing in business homepage builds and renewals.

## Tech Stack

- **Framework**: Next.js 14 (App Router) + TypeScript
- **Styling**: Tailwind CSS v3
- **Animation**: Framer Motion + Swiper.js
- **Form**: React Hook Form + Zod
- **Contact backend**: [Resend](https://resend.com)
- **Font**: Pretendard Variable (self-hosted)
- **Deploy**: Railway (Docker, Node runtime, `output: 'standalone'`)

## Local Development

```bash
# 1. Install dependencies
pnpm install

# 2. Confirm the Pretendard font file is present
#    (Already downloaded to public/fonts/PretendardVariable.woff2)
#    If missing, get the latest from:
#    https://github.com/orioncactus/pretendard/releases

# 3. Copy environment template and adjust values
cp .env.example .env.local
# Edit .env.local — at minimum, set RESEND_API_KEY or keep CONTACT_DRY_RUN=true

# 4. Run the dev server
pnpm dev
# → http://localhost:3000
```

### Useful scripts

```bash
pnpm dev          # Start dev server
pnpm build        # Production build (also generates .next/standalone for Docker)
pnpm start        # Run production build (uses PORT env var, defaults to 3000)
pnpm lint         # ESLint
pnpm typecheck    # TypeScript --noEmit
pnpm format       # Prettier write
```

## Environment Variables

See [.env.example](.env.example) for the full list. Highlights:

| Variable | Required | Notes |
|---|---|---|
| `NEXT_PUBLIC_SITE_URL` | yes | Used for canonical URLs, OG tags, sitemap |
| `RESEND_API_KEY` | yes (prod) | Get from https://resend.com/api-keys. Local dev can keep `CONTACT_DRY_RUN=true` |
| `CONTACT_FROM_EMAIL` | yes (prod) | Must be on a **verified domain** in Resend |
| `CONTACT_TO_EMAIL` | yes | Where contact submissions are delivered |
| `CONTACT_DRY_RUN` | optional | `true` to log payload instead of sending. Default `true` in `.env.example` |
| Analytics IDs | optional | Loaders no-op when ID is the placeholder. Set real IDs in Railway env to enable |

## Brand Configuration

All brand identity (name, address, business number, contact email, etc.) lives in [src/config/brand.ts](src/config/brand.ts). Edit this single file to update branding across the entire site (footer, contact page, SEO metadata, JSON-LD schema, email templates).

**Open TODOs** in `brand.ts` to fill in before launch:
- `legal.businessNumber` — 사업자등록번호
- `legal.representativeName` — 대표자명
- `phone` — 실제 전화번호
- `address.streetAddress` / `postalCode` — 실 주소
- `social.*` — 소셜 링크 (선택)

## Deploy on Railway

This project ships with a `Dockerfile` and `railway.json`. Railway auto-detects both.

1. Push to GitHub.
2. In Railway: New Project → Deploy from GitHub repo → select this repo.
3. Railway builds via the Dockerfile (multi-stage, ~70MB final image).
4. Set environment variables in Railway dashboard (copy from `.env.example`).
5. After first successful deploy, set a custom domain (e.g., `webslab.co.kr`).
6. Update `NEXT_PUBLIC_SITE_URL` in Railway env to the final domain.

### Resend domain verification (production)

To send from `hello@webslab.co.kr` instead of `onboarding@resend.dev`:

1. In Resend dashboard → Domains → Add `webslab.co.kr`.
2. Add the displayed SPF, DKIM, and return-path DNS records at your DNS provider (Gabia / Cloudflare / etc).
3. Wait for verification (usually <10 min, occasionally hours).
4. Update `CONTACT_FROM_EMAIL` in Railway to the verified address.

## Architecture

```
src/
  app/                  Next.js App Router pages
  components/
    common/             Reusable primitives (Section, Container, Heading, Button, ...)
    home/               Home page sections
    layout/             Header, Footer, FloatingBar
    contact/            Contact form (client component)
    portfolio/          Portfolio cards / detail components
    analytics/          GTM, GA4, Naver, Clarity, Hotjar, Channel.io, Mixpanel
    seo/                JSON-LD components
  config/               brand.ts, analytics.ts, site.ts
  data/                 All content as TypeScript files
  lib/                  Utilities — cn, metadata, jsonld, validators
  styles/globals.css    Tailwind + Swiper CSS + Korean line-break rules
public/
  fonts/                Pretendard Variable woff2
  images/               Place real assets here
  og/                   Open Graph images
```
